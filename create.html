1단계: URL 검증

스타트업의 웹사이트 URL을 입력하세요. URL의 유효성과 접근 가능 여부를 검증합니다.

URL 검증 기준

유효한 HTTPS 프로토콜 사용, 웹사이트 접근 가능 여부, 중복 등록 여부, 도메인 신뢰도를 확인합니다.

스타트업의 공식 웹사이트 주소를 입력하세요 (https:// 포함)

검증 결과

  • HTTPS 프로토콜 사용
  • 웹사이트 접근 가능
  • 중복 등록 아님
  • 도메인 등록 확인
  • 스팸 사이트 아님
