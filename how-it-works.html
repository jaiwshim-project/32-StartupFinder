작동 원리

StartupFinder의 랭킹 시스템이 어떻게 작동하는지, SF Score가 어떻게 계산되는지 알아보세요.

스코어 시스템

StartupFinder는 세 가지 핵심 점수로 스타트업의 인터넷 중요도와 소셜 영향력을 측정합니다.

🏆

SF Score

0 ~ 100,000

스타트업의 인터넷 중요도와 소셜 영향력을 종합적으로 반영하는 최종 점수. SF Web과 SF Social을 기반으로 산출됩니다.

🌐

SF Web

0 ~ 100,000

스타트업 웹사이트의 인터넷 중요도를 측정합니다. 인바운드 링크, 내부 링크 분포, 트래픽 등을 분석합니다.

💬

SF Social

0 ~ 100,000

스타트업의 소셜 미디어 영향력을 측정합니다. Facebook, Twitter 등의 팔로워, 참여도를 분석합니다.

SF Score 산출 공식

SF Score = f( SF Web , SF Social )

SF Score는 SF Web과 SF Social의 가중 합산으로 계산됩니다. 두 점수 모두 로그 스케일로 계산되므로, 점수가 높을수록 향상이 어려워집니다. 이는 상위 랭킹의 경쟁 강도를 반영합니다.

SF Web 측정 요소

🔗

인바운드 링크 수 & 품질

다른 웹사이트에서 스타트업 페이지로 연결되는 링크의 수와 품질. 링크 출처의 권위가 높을수록 SF Web이 높아집니다.

🔀

내부 링크 분포

같은 도메인 내 페이지 간 링크의 분포. 내부 링크가 잘 구조화될수록 SF Web이 높아집니다.

👁️

추정 방문자 & 유니크 방문자

웹사이트의 추정 방문자 수와 순수 방문자 수. 트래픽이 많을수록 높은 점수를 받습니다.

📄

페이지뷰 & 체류시간

추정 페이지뷰, 방문당 페이지뷰 수, 평균 방문 체류시간을 종합적으로 분석합니다.

SF Social 측정 요소

📘

Facebook 지표

팔로워 수, 좋아요, 댓글, 공유 수, 팬 페이지 참여자 수를 분석합니다.

🐦

Twitter (X) 지표

팔로워 수, 리트윗 품질, 즐겨찾기 수, 계정 참여도를 분석합니다.

💼

LinkedIn 지표 (예정)

회사 페이지 팔로워, 직원 수, 게시물 참여도를 분석할 예정입니다.

📺

YouTube 지표 (예정)

구독자 수, 동영상 조회수, 참여도를 분석할 예정입니다.

로그 스케일 (Logarithmic Scale)

SF Score는 선형이 아닌 로그 스케일로 계산됩니다. 이는 점수가 높아질수록 동일한 점수 향상에 더 많은 노력이 필요하다는 것을 의미합니다.

랭킹 유형

🌍

글로벌 랭킹

전 세계 모든 스타트업을 SF Score 기준으로 순위를 매깁니다. 국경을 넘어 스타트업의 위치를 확인하세요.

🏳️

국가별 랭킹

같은 국가 내 스타트업끼리 비교합니다. 해당 국가의 스타트업 생태계에서의 위치를 파악할 수 있습니다.

📍

지역별 랭킹

같은 주/도 내 스타트업끼리 비교합니다. 지역 생태계에서의 경쟁력을 확인하세요.

랭킹 참여 방법

1

스타트업 등록

웹사이트 URL과 기본 정보를 입력하여 스타트업을 등록합니다.

2

검토 & 승인

팀이 스타트업 정보를 검토하고 승인합니다. 빠른 승인도 가능합니다.

3

점수 산출

알고리즘이 SF Web, SF Social을 분석하여 SF Score를 산출합니다.

4

랭킹 반영

SF Score에 따라 글로벌, 국가별, 지역별 랭킹에 자동 반영됩니다.
